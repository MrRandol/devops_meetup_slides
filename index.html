    The toolchain

    DO YOU KNOW WHAT IT IS ? It is the concept of chaining diffent tools together to make them more powerful This is a "generic flow" => you take what you need / add some and adapt it

    - CODE: Code review, version control, merges ... - TESTS: functionnal AND performance - PACKAGE: artifacts storage, pre-release staging - RELEASE: Change management, release validation and automation - CONFIGURE: Manage and configure infrastructure / Infra As Code - MONITOR: Performance, availability, functional, end user
